# Advance Design 3D Viewer

[🇫🇷](README.md) 🇬🇧

3D viewer for Advance Design models.

<img width="1552" height="1090" alt="image" src="https://github.com/user-attachments/assets/23886c38-8ae4-4244-ab3c-52ed3355aee9" />


## Features
- 3D visualization
- Object selection
- Property extraction
- Automatic quantity takeoff
- Quantity takeoff by material
- Results on supports (ponctual, linear and planar), on walls (resultant forces) and on linear elements (diagrams)
