# Advance Design 3D Viewer

[🇫🇷](README.md) 🇬🇧

3D viewer for Advance Design models.

<img width="1552" height="1052" alt="image" src="https://github.com/user-attachments/assets/7351fd1c-cc49-476e-b69d-c81e08f9991a" />


## Features
- 3D visualization
- Object selection
- Property extraction
- Automatic quantity takeoff
- Quantity takeoff by material
- Results on ponctual support
