# Advance Design 3D Viewer

🇫🇷 [🇬🇧](README_EN.md)

Visualiseur 3D pour les modèles Advance Design.

<img width="1552" height="1052" alt="image" src="https://github.com/user-attachments/assets/7351fd1c-cc49-476e-b69d-c81e08f9991a" />

## Fonctionnalités

- Visualisation 3D
- Sélection d'objets
- Extraction des propriétés
- Métré automatique
- Métré par matériau
- Résultats sur appuis (ponctuels, linéaires et surfaciques), résultats sur les voiles (torseurs) et sur les éléments filaires (diagrammes)
- Export IFC
